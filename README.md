# LinkedIn Automator

Auto-generates LinkedIn posts per week about Data Engineering, Data Analytics, 
and Data+AI trends, plus personal portfolio project posts.

## Stack
- Python 3.11+
- SQLite for storage
- Anthropic Claude API for content generation
- Telegram bot for approval flow
- Buffer API for publishing
- Cloudflare Workers AI for images
- Excalidraw for diagrams
- GitHub Actions for scheduling
